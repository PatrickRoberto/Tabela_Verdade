Classe preposição

Gerenciar Preposicao	
	- Converter uma string em uma preposição
	- Resolver uma preposição a partir de uma lista/vetor de valores
	
	Secundarios:
		- retornar quantidade de entradas
		- Retornar lista de operações
		- Retornar preposicao partida
		
	
Tabela

Gerencia Tabela
	- Adicionar a tabela as colunas com o resultado de cada parte da preposição

	- Entradas que gera um resultado
	- Resultados com uma das entras expecificas
	
	Secundarios:
		- Devolver a lista de entradas que gera um resultado especifico
		- Devolver lista de resultados para valores fixos
